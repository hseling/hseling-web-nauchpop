Что умеет анализатор?

Анализатор разработан для обработки научно-популярных текстов на русском языке. С его помощью можно извлечь из текстов имена ученых, научные термины, опредилить тематику текстов и вывести метрики ридабилити для оценки уровня сложности текста.

Для обработки текста нажмите на кнопку Ввести текст или загрузите текст из файла с помощью кнопки Загрузить файл.
